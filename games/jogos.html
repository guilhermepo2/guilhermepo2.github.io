O que é um jogo?

Introdução

Falar de modo geral como se fazer um jogo, falar sobre OpenGL, Renderização, Física, Colisões, etc.. Falar é possível fazer jogos com puro código e OpenGL (bom para aprendizado), falar de engines gráficas (bom para a produção de jogos).

Falar sobre como fazer um primeiro jogo

Falar sobre escopo do projeto(comum exageros, ninguém vai criar o próximo GTA, seu primeiro jogo não vai ser uma obra prima, etc... etc... etc... Criar algo simples que possa ser jogado - Usar o primeiro jogo como aprendizagem - Manter ele simples (Keep it Simple Stupid) - e ficar orgulhoso dele.

Saber o que você pode e consegue fazer, dividir cada parte do jogo em pequenas tarefas...

Não reinventar a roda, criar uma engine é um trabalho para uma equipe de engenheiros e designers, engines são ótimas para a produção de jogos e possibilita até criar jogos simples sozinho, além disso, existem comunidades dispostas a te ajudar quando se usa uma determinada engine

Não ter medo de codificar - Começar com algo pequeno e manter isso simples.

Fazer o jogo baseado em sua habilidade - Bom Designer? Faça o jogo pesar mais para o lado do design - Codificador? vários jogos fizeram sucesso com um design minimalista!

Não desistir - vai ser dificil, mas lute.
