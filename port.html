Midnight Journey

Developing a game all by myself were always something I wanted to do. Midnight Journey is an Action RPG and also me trying to accomplish this ambition. As I have a programming background, little I knew about music and art, and little practice I had with design, so I started studying these aspects. It all started with a small prototype called "Zelda Souls", which led me to this first alpha.

After some time on thinking about what I wanted to do and what I was doing, I took a break from this project.

When I got back at it I decided to use Gamemaker, I took a course on Udemy and this is the final result, which is a nice base for what I want to do.

The Future

I'm currently working on the game aesthetics, changing how the main character should look like and which colors I will use, next steps are drawing animations, using them on Gamemaker and improving/working on a Game Design Document and a Story.

You can know more about Midnight Journey here and occasionally at my blog

The Ace Programmer

What is it?

The Ace Programmer came to me when I was playing... Phoenix Wright: Ace Attorney (whoah...), it is a Visual Novel Adventure Game, meaning that it is part Visual Novel (lots of dialogues and story heavy), but it also has something more interactable, on Ace Attorney you work as an... Attorney and to defend your clients you sort of have to solve puzzles. The Ace Programmer would follow a similar idea, it would have a story, it would have dialogues and a puzzle/adventure part, in this case it would be related to programming.

What I've done

I actually have done a MVP for this one, you can play it here, it has all the elements I want (except of graphics of course) which is basic dialogs and the puzzle working.

I got some pretty bad feedback on this because people where getting stuck not knowing what to do when the puzzle began, so I went back to thoughts, pen and paper to think through what I should do.

The future

I rewrote the script of the MVP (above) part, making it better and longer and I want to have it corrected by a pro. I also introduced a new character which will guide you through how to use the puzzle system, basically they will teach you how to play the game.

Next step is doing another prototype with a proper VN Engine and with temporary graphics, considering hiring a freelancer to draw the characters and consider changing to Gamemaker (as I have a license)

Pizza Clicker

What is It?

Pizza Clicker is a simple clicker, or idle, game, where you click to earn something (in this case, pizzas) and you spend this something to earn more something, the perfect addiction formula. You can be rewarded by clicking more but, usually, this kind of game turns out to be more about how to spend your somethings more efficiently and just being idle with the game open, while you earn somethings per second.

Examples of games like that are Cookie Clicker(the most famous one) and Doge Miner, there are plenty of others, you can check it.

What I've done

I did two MVPs (Minimum Viable Product) for this project, one using PhaserJS (a Javascript Game Engine) and one using plain JavaScript. The one with Phaser was more of how to display everything on screen to have an overview of how it would be aesthetically. The Javascript one is where I coded the logic, earning pizzas, buying buildings, price raising, etc... You can play the javascript version here.

The Future

I'm still thinking of which technology I should use, being the options plain javascript, html, css, PhaserJS and plain javascript with PixiJS. After that I will start drawing some sprites, there shouldn't be a lot, and then work on balancing the game progression and writing a simple story.

Jade Reno Pirates

What is it

Jade Reno Pirates is just a Rock Paper Scissors game, but Hearthstone Themed. Hearthstone once had a metagame that felt like playing rock paper scissors (Pirate Warrior, Renolock and Jade Druid), you would choose one deck and you face a mirror match or 2 other options, one of them you knew you would lose, the other one you knew you would win. So I made a rock paper scissors to remember how this metagame felt :)

What I've Done

I've drawn the basic sprites that you clicked to get to this modal page and I programmed all the game logic (which is not complex), I also developed a rock paper scissors AI to play against you :)

Why Postponed?

Because this is not an interesting game at all. But I want to get back at it one day and finish it, I just to draw a background, some UI elements and compose a simple music.

Facom Souls

What was It?

Facom Souls is a Dark Souls based combat system learning experience. I developed it on my 5th semester at University as a final project for Object Oriented Programming 2.

The purpose of the essay was to develop anything using 5 or more design patterns. I had in my mind the idea of doing something related to Dark Souls for a long time, so I tried.

What I've done

I basically programmed everything from scratch, with some help of the wonderful land of internet, of course, and used 5 different design patterns: Strategy, Mediator, Memento, Observer, State and Singleton.

Also the main character modelling was made by me, animations from Mixamo and environment/enemies are free assets from the Unity Asset Store!
