Midnight Journey

Developing a game all by myself were always something I wanted to do. Midnight Journey is an Action RPG and also me trying to accomplish this ambition. As I have a programming background, little I knew about music and art, and little practice I had with design, so I started studying these aspects. It all started with a small prototype called "Zelda Souls", which led me to this first alpha.

After some time on thinking about what I wanted to do and what I was doing, I took a break from this project. When I got back at it I decided to use Gamemaker, I took a course on Udemy and this is the final result, which is a nice base for what I want to do.

The Future

I'm currently working on the game aesthetics, changing how the main character should look like and which colors I will use, next steps are drawing animations and using them on Gamemaker.
