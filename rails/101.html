Comandos Rails:

$rails new (nome-do-projeto)

$rails server ou $rails s

$rails generate scaffold (recurso) (atributos)

- gera todos os arquivos necessários (Model, View e Controller) para que você tenha uma página simples e funcional do recurso criado com os atributos especificados, também faz alterações necessárias nas rotas.

Comandos de Banco de Dados

$rake db:create

$rake db:migrate

- sempre que um scaffold for criado (gerado ou manualmente) será necessário migrar o banco de dados para que ele possa se atualizar.

$rake db:rollback

- caso você queira desfazer a última migração, esse comando serve para isso.

Gems Auxiliares

Para instalar uma nova gem, abra o arquivo Gemfile na pasta raiz do seu projeto, depois execute bundle install no terminal

gem 'devise', '~> 3.3.0' - permite autenticação de usuários de uma maneira fácil, 3.3.0 corresponde à versão (você pode preferir outras ou podem ter outras mais recentes) - rails generate devise:install
